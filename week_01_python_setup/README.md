## Concepts

- **What Python is good for**: Python is a general-purpose programming language. It is popular because it is readable and useful for scripts, automation, websites, data analysis, machine learning, and AI tools.
- **Running a `.py` file**: A `.py` file is a Python script. Running it means asking Python to execute the instructions inside the file from top to bottom.
- **Variables**: A variable is a name that stores a value so the program can use it later. For example, `name = "Alex"` stores the text `"Alex"` inside `name`.
- **`print()`**: `print()` displays information in the terminal. It is often the first way beginners see what their program is doing.
- **`input()`**: `input()` asks the user to type something. Python receives that answer as text, even if the user typed a number.
- **Strings**: A string is text in Python. Names, cities, messages, and sentences are usually stored as strings.
- **Integers**: An integer is a whole number, such as `5`, `18`, or `2026`. Use integers when the program needs to do math with whole numbers.
- **f-strings**: An f-string lets you place variables inside text. For example, `f"Hello {name}"` builds a message using the value stored in `name`.
